HTML

  • HTML builds the frame
  • Elements can be: text, image, header, list
  • The head element contains information about the webpage. The meta element elements contains information about the page that is used by the browser
  • The body element represents the visible content shown to the user.
  • 'Styles' creates CCS file associated with the website
  • 'Base' sets the URL from the page
  • 'Link' connects an external link to the page
  • 'Script' embeds client side scrip in HTML
  • 'Charset' sets element encoding
  • 'Http-enquiv' sets HTTP header
  • 'Viewport' sets defaultview
  • 'Div' creates divison in page while 'span' creates an inline division
  • 'Ul' 'ol' 'li' - unordered list, ordered list, list
  • 'src' sets image location
  • 'A' sets links on cases
  • 'H1' 'h2' 'h3' 'h4' 'h5' h6' set heading size - h1 being the larger

CSS

  • CSS defines website style
  • The CSS file is linked with HTML within the 'head' element
  • Can define styles based on class of elements. Classes can be defined by imputing a class selector
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript helps us interact with the website
  • In JS a variable can be defined as a container and to to declare a variable, we use the 'var' keyword
  • Each line of JavaScript code ends with a semicolon.
  • The JS file is linked with HTML within the 'body' element
  • The Chrome DevTools can be accesed by right-clicking on the page and selecting 'Inspect'
  • The console.log is usually used for testing the code
  • If or else if are conditional statements that alow us to interupt control flow
  • Control flow is the order in which a computer executes code in a file. The order of code affects the outcome
  • We use the if statement when we want the code to be executed only if it it meets a conditions or is true
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result. True of False
  • Array alows us to to store a group of related data
  • Strings have to be contained within quotation marks
  • In an Array the first index stars with 0 not with 1 and increase by 1 each time.
